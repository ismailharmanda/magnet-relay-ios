# Flux Relay

Flux Relay is a portrait iPhone puzzle game about shaping magnetic fields inside a sci-fi lab. The player repositions magnets, pulses the field, and guides charged blocks into matching targets.

## Release Goal

Ship a premium-feeling native iPhone puzzle game with polished onboarding, a broad deterministic level catalog, strong motion feedback, and durable App Store release automation.

## Audience

Players who enjoy compact, replayable logic puzzles with satisfying tactile feedback: short sessions, clear goals, and increasingly clever level layouts.

## Monetization Direction

- Rewarded hint and rewarded undo hooks.
- Interstitial cadence hooks after retries or level completions.
- Future no-ads and level-pack purchase surfaces.
- v1 ships without ads, IAP, subscriptions, accounts, analytics, or tracking.

## Current Asset Stance

No Construct assets. The v1 release uses procedural native shapes/effects, Tabler MIT icons, and the narrow verified Kenney CC0 completion UI set documented in the asset catalog. Store screenshots are generated from real app screens.

The level-completion overlay uses four verified CC0 UI pieces from Kenney UI Pack: Sci-fi for glass panel, CTA, crosshair, and countdown rail treatment. Keep the imported set narrow and preserve the source/license credit dataset in the asset catalog.
